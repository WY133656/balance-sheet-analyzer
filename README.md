# 📊 资产负债表智能分析

> 上传 PDF/Excel 资产负债表 → DeepSeek 自动提取数据 → 生成专业分析报告

## 🚀 在线体验

👉 **[点击体验](https://huggingface.co/spaces/xxx/balance-sheet-analyzer)** （部署中）

或本地运行：
```bash
pip install -r requirements.txt
streamlit run app.py
```

## ✨ 功能

- 📄 支持 PDF / Excel 资产负债表上传
- 🤖 DeepSeek AI 自动理解和提取财务数据
- 📊 资产、负债、所有者权益结构化展示
- 📝 5 维度专业分析报告（资产结构 / 负债结构 / 偿债能力 / 权益分析 / 综合评级）
- 💬 基于数据的智能问答

## 🔧 技术栈

Python · Streamlit · DeepSeek · pdfplumber · openpyxl

## 💡 使用说明

1. 在 [platform.deepseek.com](https://platform.deepseek.com) 注册获取 API Key
2. 上传资产负债表
3. 点击「提取数据」→「生成报告」
4. 在问答区自由提问
